# KI-Hackathon-Energie-2026

A real-time analytics dashboard for a DC fast-charging hub, built at the KI-Hackathon Juni 2026.

---

## Architecture

```
EnergieHackathon/
├── backend/                        # FastAPI (Python) — data cleaning, API, pricing logic
│   ├── api.py                      # REST API endpoints
│   ├── clean_data.py               # Cleans raw Excel files → data/clean/
│   ├── strategy.py                 # Dynamic pricing & offer logic
│   ├── analytics.py                # Peak shaving, load shift analysis
│   ├── attribution.py              # Notification attribution & conversion tracking
│   ├── notify.py                   # Push notification logic
│   ├── models.py                   # Data models
│   ├── config.py                   # Tariff & system configuration
│   ├── data/
│   │   ├── *.xlsx / *.pdf          # Raw input files (read-only)
│   │   └── clean/                  # Generated by clean_data.py
│   │       ├── master_15min.csv    # Load profile + spot prices (15-min grid)
│   │       └── sessions.csv        # Cleaned charging sessions
│   └── frontend/                   # Static HTML apps served at /ui
│       ├── dashboard.html          # Operator dashboard (static)
│       └── app.html                # Driver app (static)
│
├── frontend_client/                # React + Vite — operator analytics dashboard
│   ├── dist/                       # Production build output
│   └── src/
│       └── components/
│           ├── Dashboard.jsx       # Main dashboard layout & state
│           ├── CombinedChart.jsx   # Combined overview chart
│           ├── AnalyticsChart.jsx  # Individual metric charts
│           └── api.js              # Fetches & transforms data from backend
│
└── user_app/                       # React + Vite — driver-facing app
    └── src/
        └── components/
            ├── ChargingDashboard.jsx
            ├── PriceChart.jsx
            ├── PriceList.jsx
            ├── ProfileManager.jsx
            └── StationManager.jsx
```

**Ports:**
- Backend API: `http://localhost:8001`
- Frontend (React): `http://localhost:5173`
- Static driver app: `http://localhost:8000/ui/app.html`
- API docs: `http://localhost:8000/docs`

## Setup & Running

### Prerequisites

- Python 3.10+
- Node.js 18+

---

### 1. Install backend dependencies

```bash
cd backend
pip install -r requirements.txt
```

### 2. Clean the data

Must be run once before starting the backend (and again if raw data changes):

```bash
cd backend
python3 clean_data.py
```

### 3. Start the backend

```bash
cd backend
uvicorn api:app --reload --port 8000
```

### 4. Start the frontend

In a separate terminal:

```bash
cd frontend_client
npm install   # first time only
npm run dev
```

Open `http://localhost:5173` in your browser.
